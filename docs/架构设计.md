## 主要接口定义

### Writer

```golang
type Writer interface {

	// Write 根据写请求执行写操作
	Write(arg WriteRequest) (bool, error)

	// Name 返回 Writer 的名称
	Name() string
}
```

### Reader

```golang

type Reader interface {

	// Read 根据读请求执行查询操作
	Read(query Query) (interface{}, error)

	// Name 返回 Reader 的名称
	Name() string
}
```

### Sinker

```golang

type Sinker interface {

	// Send 负责发送数据
	Send(arg interface{})

	// Receive 负责接收一个数据
	Receive() interface{}

	// Name 返回 Sinker 的名称
	Name() string
}
```
